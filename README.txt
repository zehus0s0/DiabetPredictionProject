FILES IN THE DISK:
- diabet_model.py: Main Python script containing model training and evaluation code
- app.py: Streamlit app for user interface
- diabetes_model.pkl: Trained Random Forest model
- diabetes_scaler.pkl: StandardScaler object used for feature scaling
- diabetes_medians.pkl: Median values used to impute missing data
- sample_input.csv: Sample clinical input data
- sample_output.txt: Sample prediction output
- requirements.txt: Python dependencies
- README.txt: This file

HARDWARE REQUIREMENTS:
- OS: Microsoft Windows 11 Home (64-bit)
- CPU: AMD Ryzen 7 5700U with Radeon Graphics, 8 cores
- RAM: 8 GB or higher
- Storage: At least 300 MB free disk space
- GPU: Not required (project runs on CPU)
- Internet: Required for accessing Streamlit web interface (optional)
